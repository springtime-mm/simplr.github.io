git

发表于 2023-06-05

使用git-bash 输入命令：

git cat-file -p <版本号> 根据版本号查看提交信息

git -v查看软件版本

git init初始化仓库

git clone <url>克隆远程仓库到本地

git clone <url> <自定义文件夹名>克隆远程仓库到本地并自定义名字

git config <配置名称> <配置值>git配置

git config user.name 用户名 配置 username

git config user.email 邮箱地址 配置 email

git config --global <配置名称> <配置值>git 进行全局配置

git statusgit 暂存区的状态

git add <文件名> 从工作区添加文件到暂存区

git rm --cached <文件名> 从暂存区将文件移回工作区

git commit -m "<提交的备注消息>" 将暂存区的文件提交到存储区

git log 查看提交历史

git log --oneline 以更简洁明了的方式显示提交历史

git restore <文件名> 恢复尚未提交的误删除的文件

git reset --hard <7位版本号> 将当前的版本库重置到之前的库

git revert <需要恢复的版本号的下一个版本号> 在不丢失之前提交历史的情况下，恢复到之前的版本库

git branch <分支名> 新增分支

git branch -v 查看有多少分支

git branch -d <分支名> 删除分支

git checkout <分支名> 切换分支

git checkout -b <分支名> 创建分支并切换过去

git merge <分支名> 合并分支

    若有冲突，需要解决冲突。

git tag 查看当前的标签

git tag <标签名> <版本号> 为提交记录添加tag

git log <标签名> 通过标签名查看提交历史

git checkout -b <标签名> 以当前标签的提交创建一个分支

git remote add <origin> <ssh地址> 添加名为origin的远程地址

git remote remove <origin> 删除远程地址

git remote rename <origin> 修改名称

git remote -v 查看远端分支

git push <origin> 推送到名为origin的远程仓库

git pull <origin> 拉取名为origin的远程仓库
